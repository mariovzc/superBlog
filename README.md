# SuperBlog



Desafio de MakeitReal.

Funcionalidades:

  - Registrarse
  - Ingresar
  - Listar Posts
  - Ver Post
  - Crear Post
  - Editar Post
  - Eliminar Post
  - Comentar Post



### Gemas

* [Rspec]
* [BetterErrors] 
* [Shoulda]
* [Railroady]
* [Sorcery]


### Installation


```sh
$ cd superBlog
$ bundle install
$ rake db:migrate && rake db:seed
$ rails s
```
